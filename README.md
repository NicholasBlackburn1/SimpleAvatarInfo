### Simple avatar info 
- this mod allows the user to view and clone / rip the avatar from vrchat 


### How to use 
- go into vrchat with this mod in the mods folder and click on the left wing menu and one of the 4 buttons
- avatar info
- avatar cloning
- mod info
- Download path
